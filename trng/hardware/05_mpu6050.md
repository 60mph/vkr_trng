# Схема №5 — MPU-6050 (модуль GY-521)

## BOM

- Модуль GY-521 с MPU-6050.
- 4× провода Dupont (M-F): VCC, GND, SDA, SCL.

## Подключение

| GY-521 | Arduino Nano |
|--------|--------------|
| VCC    | 5V (через LDO модуля становится 3.3 В) |
| GND    | GND          |
| SCL    | A5           |
| SDA    | A4           |
| AD0    | GND (адрес 0x68) — пин на модуле уже подтянут к GND, можно не трогать |
| INT    | (не подключаем) |

## Особенности подключения

- Подтяжки I²C линий (SDA/SCL) к 3,3 В уже есть на модуле — **не добавляйте
  свои к 5 В**.
- Если нужна максимальная скорость опроса, прошивка использует I²C 400 кГц
  (Fast Mode); большинство модулей GY-521 это поддерживают.

## Расположение

- Модуль кладётся **горизонтально** на стол. Любая вибрация снижает качество
  бит из-за коррелированного "сигнала".
- Дайте чипу прогреться 3–5 минут перед записью данных.

## Проверка

В Arduino IDE есть Serial Monitor (115200) — можно временно прошить любой
демо-скетч из библиотеки `MPU6050_tockn` или подобной, чтобы убедиться, что
устройство отвечает по I²C на адресе 0x68. После этого прошиваем
`05_mpu6050/05_mpu6050.ino` и переключаемся на 1 Мбод для захвата.
