# Источник №3 — Лавинный пробой BE-перехода BJT

**Раздел дипломной работы:** 2.1, "Дробовой шум" + "Лавинный шум".

## Физический принцип

При обратном смещении эмиттер-база переход кремниевого BJT (NPN) пробивается
обычно при U_BE_breakdown = 6...9 В. В отличие от стабилитронов, конструкция
BJT не оптимизирована под пробой, поэтому ток пробоя крайне нестабилен и шумен.
Это классический "трюк" любителей радиоэлектроники — самый дешёвый источник
истинной случайности с амплитудой шума, превосходящей тепловой на 30–50 дБ.

Ожидаемый спектр — белый до МГц-диапазона. Для данной схемы (ATmega328P,
fs=76 квыб/с) этой полосы более чем достаточно.

## Установка эксперимента

| Параметр                  | Значение |
|---------------------------|----------|
| Дата                      |          |
| Транзистор (тип, образец) | 2N3904 #1|
| U_питания, В              | 9 (Vin)  |
| R_emitter, кОм            | 470      |
| I_breakdown (расчётный)   | 1...10 мкА |
| Усиление LM358(A)×(B)     | 1000 × 1 |
| Опорное ADC               | AVCC     |
| Частота дискретизации, Гц | 76923    |
| Длительность записи, мин  |          |

Аппаратная часть — `hardware/03_bjt_avalanche.md`.

## Замечания

- **Селекция образцов**: разные экземпляры одной и той же модели (даже из
  одной партии) пробиваются при чуть разных напряжениях и шумят по-разному.
  Имеет смысл прогнать pipeline.py для 2–3 экземпляров и выбрать "лучший".
- **2N3904** обычно пробивается стабильно при 7,5–9 В. **2N2222 / BC337**
  — около 6,5–7,5 В. Для 5 В питания возможен только дробовой шум на BE
  без лавинного пробоя — амплитуда сильно меньше.
- Коллектор оставляем в воздухе (NC). При подключении ток сильно меняется,
  спектр уходит в розовый шум.

## Команда запуска

```bash
python scripts/pipeline.py --source 03_bjt_avalanche --port /dev/ttyUSB0 \
    --bytes 10485760 --bits 1 --sts ./nist_sts/sts-2.1.2
```
