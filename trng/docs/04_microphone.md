# Источник №4 — Акустический шум (KY-037)

**Раздел дипломной работы:** 2.3, "Микрофоны и пьезоэлектрические датчики".

## Физический принцип

Электретный микрофон даёт суммарный сигнал из:

1. **Окружающий акустический фон** (вентиляция, голоса, шаги).
2. **Тепловой шум** мембраны и предусилителя на плате KY-037.
3. **Дробовой шум** в JFET-предусилителе микрофона.

Результат — нестационарный, цветной (розоватый) шум. **Сырая** энтропия
плохая (низкая равномерность, высокая корреляция), но после извлечения LSB и
Von Neumann становится приемлемой.

## Команда запуска

```bash
python scripts/pipeline.py --source 04_microphone --port /dev/ttyUSB0 \
    --bytes 10485760 --bits 1 --sts ./nist_sts/sts-2.1.2
```

## Особенности обработки

Для акустики **рекомендуется** использовать `mix.py --mode sha256` после
extract_bits, иначе спектральная неравномерность убивает многие тесты NIST.

```bash
python capture/mix.py --mode sha256 \
    --in data/processed/04_microphone/run_001_lsb1.bin \
    --out data/processed/04_microphone/run_001_sha.bin
```
