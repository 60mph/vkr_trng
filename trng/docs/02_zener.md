# Источник №2 — Шум стабилитрона 5,1 В (лавинный/Зенера)

**Раздел дипломной работы:** 2.1, "Лавинный шум".

## Физический принцип

При обратном включении стабилитрона в области пробоя ток состоит из двух
вкладов:

- **Зенеровский пробой** (U_z ≤ ~5 В): туннельный, плотность шума умеренная.
- **Лавинный пробой** (U_z > 5–6 В): ударная ионизация, плотность шума выше
  и характеризуется коэффициентом избыточного шума F:

\[
S_I(f) = 2qIM^2F
\]

5,1 В стабилитрон — это переходная зона, поэтому ток пробоя нестабилен и
"бурлит" с очень широкой полосой (десятки МГц). Для ADC 76 квыб/с на A0
этого с запасом хватает: вся полоса будет почти идеально белой.

## Команда запуска

```bash
python scripts/pipeline.py --source 02_zener --port /dev/ttyUSB0 \
    --bytes 10485760 --bits 1 --sts ./nist_sts/sts-2.1.2
```

## Результаты (заполняются после прогона)

- **PSD:** ожидаем плоский в полосе дискретизации, насыщенный относительно
  теплового шума (амплитудный SNR на 10–20 дБ выше).
- **Гистограмма:** широкая, близкая к равномерной после AC-связи; смещение
  даст χ²-fail (это **нормально** для сырых данных, исправляется LSB-extract +
  Von Neumann).
- **NIST STS:** ожидаем PASS по большинству тестов уже после LSB1+VN; если
  есть FAIL по Frequency / Cumulative Sums — увеличить число усилительных
  каскадов или применить мягкое хеширование (`mix.py --mode sha256`).
